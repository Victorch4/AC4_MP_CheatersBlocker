This is a tool for easily blocking cheaters in Assassin's Creed IV Black Flag.

You can use hotkeys to block cheaters even if this tool is minimized and the game is running.
For convenience, all actions are voiced using AI.

F1 - Block/Unblock all traffic for selected IP

F2 - Block/Unblock IN traffic for selected IP

F2 - Block/Unblock OUT traffic for selected IP
